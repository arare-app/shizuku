Shizuku

Shizuku is a mod for Bondage Club that adds some cheat-oriented features to the game.

Install

You can install the mod by using Tempermonkey or the manual bookmark method.

NOTE:

Also, as this scripts requires BCX load first, you need to install it too.

1) Using Tempermonkey

Make sure you have Tempermonkey installed and enabled in your browser. Then, click here to install the mod.

2) Manual bookmark method

Create a new bookmark in your browser and paste the following code in the URL field:
